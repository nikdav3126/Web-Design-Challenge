Summary: City Latitude and Weather

This project was a chance to see how a city's location, specifically a city's latitude, would affect the weather experienced. While doing the analyses we wanted to see if a city's relation to the equator had any bearing on the weather as well.

In order to run our queries we first needed to pull data from OpenWeatherMap API to assemble our dataset on over 500 cities. Once we had the dataset we were able to use Matplotlib to plot our various aspects of the cities. These aspects included temperature, humidity, cloudiness, and wind speed. This site was created to show the visualizations created in Matplotlib, as well as descriptions of any trends or correlations noted.
